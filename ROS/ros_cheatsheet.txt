# ROS cheat sheet
https://subscription.packtpub.com/book/hardware_and_creative/9781782175193/1/ch01lvl1sec15/ros-commands-summary

catkin_make         build your ROS source code.
roscore             start ROS running on a terminal
rosnode             Shows info about nodes
rostopic            Shows info about topics/channels
                    rostopic [command] [channel]
                        command = echo, info, type

rosrun [package] [executable/python script] Runs nodes
roslaunch [package] [launch file]           Kicks off .launch file
